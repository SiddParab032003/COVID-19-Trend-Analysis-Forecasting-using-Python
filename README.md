# COVID-19-Trend-Analysis-Forecasting-using-Python
Built a data science project to analyze COVID-19 trends using Python, visualized insights with Plotly, and applied Prophet for time-series forecasting to predict future cases and understand pandemic growth.

##  Project Overview
This project analyzes global COVID-19 data to identify trends in confirmed, recovered, death, and active cases, and forecasts future cases using time-series modeling.

The objective is to understand pandemic growth patterns and generate predictive insights using data science techniques.

---

##  Dataset Description
The dataset contains daily global COVID-19 records:

- Date: Daily case records  
- Country/Region: Country name  
- Confirmed: Total confirmed cases  
- Deaths: Total deaths  
- Recovered: Total recovered cases  
- Active: Active cases  

---

##  Exploratory Data Analysis (EDA)

Key analysis performed:

- ✔ Data cleaning and preprocessing  
- ✔ Handling missing values  
- ✔ Aggregation of global data  
- ✔ Time-series trend analysis  

###  Key Insights:
- Confirmed cases show **exponential growth**, especially after March 2020  
- Deaths increase steadily but slower than confirmed cases  
- Recovery cases improve over time  
- Active cases highlight **healthcare system pressure**  

---

##  Trend Analysis

- Rapid increase in cases indicates global outbreak acceleration  
- Gap between confirmed and recovered cases reduces over time  
- No early signs of flattening in the curve  

---

##  Visualization

##  Time Series Forecasting

###  Prophet Model
- Used for forecasting future COVID-19 cases  
- Captures trend effectively  
- Predicts next 7 days of confirmed cases  

---

##  Forecast Insights

- Continuous upward trend in predicted cases  
- No immediate decline observed  
- Narrow confidence intervals indicate reliable short-term predictions  

---

##  Forecast Results

- Cases expected to increase steadily  
- Model aligns closely with actual data  
- Shows strong trend capturing ability
